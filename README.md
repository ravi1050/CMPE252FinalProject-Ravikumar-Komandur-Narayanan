# CMPE 252 AI MS Project

This repository contains my MS AI coursework project materials for CMPE 252, focused on parameter-efficient fine-tuning and comparative evaluation of modern adaptation methods.

## Scope

The project explores:

- baseline model evaluation
- LoRA fine-tuning in medical and legal domains
- cross-domain transfer behavior
- rank and training-size ablations
- comparison against full fine-tuning
- alternate PEFT strategies including AdaLoRA, IA3, and QLoRA

## Repository Layout

- `notebooks/`: experiment scripts and model evaluation workflows
- `report/`: final report source and bibliography
- `results/`: experiment summaries
- `00_Project_Overview.md` to `06_Experiment_Plan.md`: planning and supporting documentation

## Notes

This repository is organized to keep proposal material, experiment code, and report assets together for submission and future extension.
